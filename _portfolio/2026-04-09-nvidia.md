## Recolección y limpieza de datos

Los datos fueron obtenidos de [Yahoo Finance](https://finance.yahoo.com/quote/NVDA/financials/), página que ofrece una enorme cantidad de información sobre empresas de distintos giros.

Anteriormente, la página contaba con un botón que permitía descargar la información financiera de cualquier empresa en un archivo de Excel. Sin embargo, esa funcionalidad fue retirada y al momento de copiar los datos a una hoja de Excel, estos se pegaban en una sola columna.

Por esta razón tuve que escribir un pequeño _script_ en **Python** que me permitiera visualizar los datos como una tabla. Añado aquí el _script_ dada su reducida longitud.

```python
from openpyxl import load_workbook, Workbook


# Variables para el tamaño de la tabla
NUM_ENCABEZADOS = 6
NUM_FILAS = 210

wb = load_workbook("Data.xlsx")
hoja = wb.active

encabezados = []
indices = []
valores = []

for i in range(1, NUM_ENCABEZADOS + 1):
    encabezados.append(hoja.cell(row=i, column=1).value)

for i in range(NUM_ENCABEZADOS + 1, NUM_FILAS + 1):
    if (i - 1) % NUM_ENCABEZADOS == 0:
        indices.append(hoja.cell(row=i, column=1).value)
    else:
        valores.append(hoja.cell(row=i, column=1).value)

temp = []
data = {}

for i in range(len(valores)):
    if i % 5 == 0:
        data[f"{i//5}"] = temp
        temp = []

    temp.append(valores[i])

for i in range(1, NUM_ENCABEZADOS + 1):
    hoja.cell(row=1, column=i, value=encabezados[i - 1])

for i in range(2, len(indices) + 2):
    hoja.cell(row=i, column=1, value=indices[i - 2])

for i in range(1, len(data)):
    for j in range(len(data[f"{i}"])):
        hoja.cell(row=i + 1, column=j + 2, value=data[f"{i}"][j])

wb.save("Clean_data.xlsx")

```
